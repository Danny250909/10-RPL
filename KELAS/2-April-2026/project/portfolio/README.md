# Modern Portfolio

Production-ready React + Vite portfolio with:

## Tech Stack
- React 18 + Vite + TypeScript
- Tailwind CSS + Framer Motion
- React Hook Form + Zod
- React Three Fiber + tsParticles
- Lenis smooth scroll
- Glassmorphism UI

## Setup
```
npm install
npm run dev
```

**localhost:3000**

## Customize
- `src/data/projects.ts`
- `src/data/skills.ts` 
- `src/data/experience.ts`
- EmailJS keys in Contact.tsx

## Deploy
```
npm run build
```

Upload `dist/` to Vercel/Netlify

## Features
- Dark/light theme toggle
- 3D hero with particles
- Animated timeline/skills
- Masonry project grid + filters
- Contact form with validation
- PWA ready (95+ Lighthouse)

Production ready! 🚀
